Webpack build tools

Besides Calibri you can use a webpack build tools

What you can use during development

All files that should be processed with webpack stored in the /resources
Processed files stored in the /storage/app/assets

To use built assets create storage folder in the public and link /storage/app to /public/storage/app

Major features

  • HTML preprocessor: Pug
  • CSS preprocessor: Sass (Scss)
  • JS preprocessor: TypeScript

Scss and TypeScript files available only with pug compiled views.
Files, processed with webpack has .compiled suffix

You are free to use assets without webpack by creating them anywhere you want inside the /public

Additional packages:

  • Tailwind (already integrated with PostCSS)
  • Bootstrap 5 (added as npm package)
