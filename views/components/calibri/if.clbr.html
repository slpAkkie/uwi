##If Using condition output

Use ##if directive to set conditional output

Examples

##if(true) This text will be visible #endif
##if(true, 'This text will also be visible')
##if(false, 'But it won\'t be visible')

Result

#if(true) This text will be visible #endif
#if(true, 'This text will also be visible')
#if(false, 'But it won\'t be visible')

Unfortunately ##else directive not supported yet.

And also the content will still be executed and parsed even if a statement in the ##if directive equals false.
